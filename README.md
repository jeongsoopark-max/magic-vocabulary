# magic-vocabulary
중고등학생을 위한 영단어 연습 프로그램 📖 | Vocabulary practice app for middle &amp; high school students
"이 프로그램은 중고등학생들의 내신 및 수능 대비를 돕기 위해 만들어진 맞춤형 영단어 학습 앱입니다. 사용자는 초등, 중등, 고등 수준별로 단어를 선택하여 학습할 수 있으며, 5지선다 객관식과 주관식 퀴즈를 통해 실력을 점검할 수 있습니다. 특히 헷갈리는 단어는 구글 AI(Gemini)가 실전 시험에 나올 법한 맞춤형 예문과 해석을 즉시 생성해 주어 완벽한 암기를 돕습니다."
